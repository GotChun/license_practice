특징
- 무결성 : 동일한 내용에 대해서 서로 다른 데이터가 저장되는 것을 허용치 않음
- 일관성 : CRUD 후에도 저장된 데이터가 변하지 않게 함
- 회복성 : 에러가 발생 시 특정 상태로 복구되어야 함
- 보안성 : 불법적인 변경,손실로부터 보호되어야 함
- 효율성 : 저장 공간 등이 최적화되어 사용자에게 효울적으로 사용성을 줌

계층형 데이터베이스 : 트리구조 , 계층화하여 관리하는 시스템
망형 : 데이터 구조를 네트워크상의 망 형태로 논리적으로 표현한 데이터 시스템
관계형 : 보편적으로 사용하는것, 테이블의 일부를 다른 테이블과 관계시켜서 관리하는 시스템

메타데이터  = 데이터사전 (Data Dictionary)
정규화 : 중복제거,데이터 무결성 유지
비정규화 : 데이터베이스 성능을 높이기 위해 의도적으로 중복 허용, 조회 성능 최적화
